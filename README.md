# 🏠 우리집 집안일 트래커

가족 모두가 함께 사용하는 집안일 관리 & 용돈 정산 웹앱입니다.

## 주요 기능

**집안일 기록** — 가족 구성원을 선택하고 집안일 버튼을 누르면 자동으로 금액이 누적됩니다.

**달력 보기** — 월별 달력에서 날짜별 활동을 한눈에 확인. 날짜를 탭하면 누가 무엇을 했는지 세부 내역이 팝업으로 표시됩니다.

**정산 기능** — 누적된 금액을 정산하면 기록이 초기화되고, 이전 정산 내역도 보관됩니다.

**자유로운 커스텀** — 가족 구성원 추가/삭제, 집안일 종류와 금액을 자유롭게 설정할 수 있습니다.

**공유 데이터** — 같은 링크로 접속한 모든 가족이 동일한 데이터를 보고 수정할 수 있습니다.

## 기술 스택

- **React** (JSX)
- **Tailwind-free** inline styling
- **Persistent Storage API** — 공유 데이터 저장
- **Noto Sans KR** — 한글 웹폰트

## 사용 방법

1. Claude.ai에서 이 파일을 Artifact로 실행
2. 앱 화면 우측 상단 `⋯` → `Publish`로 링크 생성
3. 가족 카톡방에 링크 공유
4. 가족 모두 같은 링크로 접속하여 사용!

## 화면 구성

- **🏠 홈** — 누적 총액, 구성원별 현황, 집안일 기록, 최근 활동
- **📅 달력** — 월별 캘린더, 날짜별 세부 내역, 월간 요약
- **💰 정산** — 구성원별 정산 내역, 정산 실행, 이전 정산 기록
- **⚙️ 설정** — 가족 구성원 관리, 집안일 목록 관리
