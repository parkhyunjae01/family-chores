# 🏠 우리집 집안일 트래커

가족 모두가 함께 사용하는 집안일 관리 & 용돈 정산 웹앱입니다.
Firebase 실시간 데이터베이스로 가족 누구나 어디서든 같은 데이터를 보고 수정할 수 있어요.

## 주요 기능

**집안일 기록** — 가족 구성원을 선택하고 집안일 버튼을 누르면 확인 팝업 후 금액이 누적됩니다.

**달력 보기** — 월별 달력에서 날짜별 활동을 한눈에 확인. 날짜를 탭하면 누가 무엇을 했는지 세부 내역이 표시됩니다.

**📊 리포트** — 월별 공정성 점수, MVP, 기여도 차트, 요일별 활동량, 연속 기록 등 가족 집안일 패턴을 분석합니다. 이전 달 리포트도 조회 가능합니다.

**정산 기능** — 누적된 금액을 정산하면 홈 화면 금액은 초기화되지만, 기록은 달력과 리포트에서 계속 볼 수 있습니다. 정산된 항목은 ✅ 표시로 구분됩니다.

**실시간 공유** — Firebase Realtime Database 기반으로, 아빠가 기록하면 엄마 폰에서도 즉시 반영됩니다.

**자유로운 커스텀** — 가족 구성원 추가/삭제, 집안일 종류와 금액을 자유롭게 설정할 수 있습니다.

## 기술 스택

- **React 18** — CDN 기반, 빌드 도구 없이 단일 HTML 파일로 동작
- **Firebase Realtime Database** — 가족 간 실시간 데이터 동기화
- **GitHub Pages** — 무료 정적 호스팅
- **Noto Sans KR** — 한글 웹폰트

## 화면 구성

- **🏠 홈** — 누적 총액, 구성원별 현황, 집안일 기록, 최근 활동
- **📅 달력** — 월별 캘린더, 날짜별 세부 내역, 월간 요약
- **📊 리포트** — 공정성 점수, MVP, 기여도 차트, 요일별 활동량
- **💰 정산** — 구성원별 정산 내역, 정산 실행, 이전 정산 기록
- **⚙️ 설정** — 가족 구성원 관리, 집안일 목록 관리, 연결 상태
